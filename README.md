# 🧮 Matrix Calculator (NumPy)

A simple command-line Matrix Calculator built using Python and NumPy.  
This project allows users to perform basic matrix operations interactively.

---

## 🚀 Features

- Matrix Input (row-wise)
- Transpose of a matrix
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- User-friendly CLI menu

---

## 🛠️ Technologies Used

- Python
- NumPy

---

## 📌 How It Works

1. User enters the size of the matrix
2. Inputs matrix values row-wise
3. Selects an operation from the menu
4. Program performs the operation and displays result
